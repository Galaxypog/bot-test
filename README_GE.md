Kanal Name
Verwendbare Variablen 
```
{name}                           : Bot's display name
{id}                             : Bot's ID
```
Sie können diese als Befehlskanal verwenden 
```
Test-Bot1-command-channel
Test-Bot2-command-channel
```
Wenn die Einstellung Standard ist 
```
{name}-command-channel
```
und Bot heißt
```
Test Bot1
Test Bot2
```

IP
Verwendbare Variablen  
```
{ip}                             : Default IP
```

Match-Methode  
```
full     : Perfect matching
contains : Contains it
starts   : Stats with it
ends     : Ends with it
```

Variante  
```
Muster / numerisch / Kleidungsfarbe / Trikotfarbe / Teile / progressiv / Partikel / Material / Emissionsmittel
In der Regel werden häufig materielle, progressive Teile verwendet
Zum Beispiel ist lila Schädel Soldat
Kleidungsfarbe 1
jersey_color werden für Fußballfelle verwendet
```

```
Zum Beispiel sind 8ball vs Scratchs Panneninformationen Erleuchtung
Staffel (In Kapitel 2) / Nummer
```
